# lounge-waiting
PWA로 만들어진 신세계백화점 VIP 라운지 웨이팅 모아보기 웹앱입니다.  
해당 서비스는 Firebase Hosting을 사용하여 호스팅되어 있습니다.

## 서비스 이용하기
### 웹
